# V2X-SenseComm Website

This repository contains the static HTML/CSS project homepage for **V2X-SenseComm**.

## Publish with GitHub Pages

1. Put this repository on GitHub.
2. Go to `Settings -> Pages`.
3. Select `Deploy from a branch`.
4. Choose `main` and `/docs`.
5. Save and wait for deployment.

## Edit

- Main page: `docs/index.html`
- Style file: `docs/styles.css`
- Figures: `docs/assets/figures/`
- Images: `docs/assets/images/`
